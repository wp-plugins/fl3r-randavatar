Creates a unique avatar for each commenter based on email address, with godzillion of generated avatars!

== Description ==

This plugin provides a large randomly assembled avatar for each user based on their email address. When a user comments on the site, the plugin generates a unique avatar based on his email. There are an impressive number of avatars that can be generated, and it increases at each update of the plugin! 

== Installation ==

1. Download and unzip the plugin file.
2. Upload "fl3r-randavatar" folder into your /wp-content/plugins/ directory.
3. Make sure the 'avatar' folder is [writable].
4. Go to the Plugins section of your WordPress admin panel and activate the FL3R Randavatar.
5. And that's all folks!
